# GoToMad

## Présentation

GoToMada est une agence de voyage spécialisée dans l'organisation de séjours à Madagascar.

Le projet consiste à concevoir une plateforme web permettant aux voyageurs de découvrir et réserver des séjours organisés selon leurs préférences et leur budget.

---

## Contexte

- Gestion des offres touristiques actuellement peu digitalisée.
- Absence de plateforme web interactive.
- Volonté de centraliser les informations et les séjours proposés.

---

## Objectifs

- Présenter les destinations touristiques de Madagascar.
- Proposer des séjours organisés clés en main.
- Faciliter l'organisation des voyages.
- Valoriser le patrimoine culturel et naturel malgache.
